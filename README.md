# 💥 BOOM TETRIS (Explosion Tetris) 開発中

ブラウザで遊べるカオス系テトリスゲームです。
通常のテトリスに加えて、**時間経過でブロックが爆発する**というユニークな要素が特徴です。

---

## 🎮 特徴

* ⏱ ブロックは約10秒後に爆発
* 💥 爆発で周囲のブロックも巻き込み消滅
* 🤖 空からランダムなお邪魔ミノが降ってくる
* 📈 レベルに応じて難易度アップ
* ✨ パーティクルによる爆発演出

---

## 🚀 使い方

### 1. ファイルを保存

```bash
tetris.html
```

### 2. ブラウザで開く

ダブルクリック、またはブラウザにドラッグ＆ドロップするだけでOK

---

## 🧠 ゲームのルール

* ブロックは設置後、一定時間で爆発します
* 爆発は周囲1マスにも影響します
* ラインを揃えるとスコア加算
* レベルが上がると落下速度と難易度が上昇
* 画面上部にブロックが到達するとゲームオーバー
