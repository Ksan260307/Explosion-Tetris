# 💣 Explosion Tetris: UCD-F Core Simulation
**制約下動的状態遷移システムによるテトリスの混沌的進化**

『Explosion Tetris』は、単なるパズルゲームではありません。極端なパフォーマンス制約と決定論的動態を定義した設計思想 **「Universal Constrained Dynamics Framework (UCD-F)」** に基づき構築された、高精度な状態遷移シミュレーターです。

クラシックな落ち物パズルの構造に、環境エントロピーから生成される揮発的な「爆弾」と、時間を巻き戻す「因果逆転」のメカニズムを融合させています。

---

## 🏗 システムアーキテクチャ (UCD-F Spec)

本シミュレーションは、以下の高度な制約技術によって駆動しています：

* **Bit-Packed SoA (Structure of Arrays):** 盤面のすべての状態は単一の `Uint32Array` 内で管理。色、爆弾の導火線、消去フラグ、座標オフセットを32bit整数にパッキングし、コアロジック中のメモリアロケーションをゼロ化（Zero-Allocation）しています。
* **16.16 固定小数点演算:** パーティクル等の物理演算から浮動小数点を排除。デバイス間の演算誤差による非同期を遮断し、完全な決定論的挙動を保証します。
* **Entropy Harvesting (環境ノイズの収穫):** デバイスのノイズやユーザー入力をシステム内のエントロピー（E2/E3指標）として収穫し、爆弾の生成率や「疲労（VIF Fatigue）」に直接反映させます。
* **Entropy Reversal (時間逆行):** 最大5秒前までのシステム状態をリングバッファに保持。因果を遡ることで、システム崩壊（ゲームオーバー）を回避可能です。

## 💣 コアダイナミクス

### ■ 時限爆弾エンティティ
特定のミノは、不安定な「時限爆弾」を含んだ状態で実体化します。これらは固有の半減期（通常10秒 / 危険状態5秒）を持ち、ライン消去によって解除されない場合、運動エネルギーの爆発を引き起こして周囲のブロックを破壊し、強力な「E3ノイズ（お邪魔ブロック）」を生成します。

### ■ VIF Dynamics (速度・強度・疲労)
* **Velocity (速度):** レベルの上昇に伴い、エンティティの落下速度が指数関数的に向上。
* **Intensity (強度):** ライン消去やコンボ（REN）に応じて、獲得スコアと視覚的フィードバックの強度がスケーリング。
* **Fatigue (疲労/DANGER):** 爆発の許容限界を超えるとシステム疲労（DANGER）が蓄積。限界に達すると **ZERO-LOCK** (システム停止) となります。

### ■ Mega Disarm (臨界点解除)
爆発まで残り1秒以下の臨界状態で爆弾を解除すると **Mega Disarm** が発動。盤面上の全アクティブエンティティを一掃し、莫大なスコアボーナスを獲得します。

## 🕹 操作インターフェース

### PC 操作
| アクション | デフォルトキー |
| :--- | :--- |
| 左右移動 | `←` / `→` |
| 右回転 | `↑` |
| ソフトドロップ | `↓` (押しっぱなしで高速) |
| ハードドロップ | `Space` (衝撃波発生) |
| ホールド | `Shift` |
| **時間逆行 (E-REV)** | **`Z` (長押し)** |
| 一時停止 (Suspend) | `P` |
| 設定 (Configure) | `O` |

### モバイル / タッチ操作
* **タップ:** エンティティの回転
* **左右スワイプ:** 水平移動
* **下スワイプ:** ソフトドロップ（長押し対応）
* **上フリック:** ハードドロップ
* **2本指タップ / HOLDボタン:** ホールド
* **E-REVボタン長押し:** 時間逆行

## 🛠 環境設定パラメータ
`UCD-F PARAMS` コンソールから、SE音量、BGM（ドローンシンセ）、ロックダウン猶予時間、スワイプ感度、難易度（VIFスケーリング）を調整可能です。設定値およびハイスコアは、ブラウザのローカルストレージに永続化されます。
